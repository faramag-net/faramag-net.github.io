Término	Definición
Artículo	Todo elemento administrado por el negocio.
Transacción	Todo hecho de negocio registrado.
Movimiento	Cambio físico derivado de una transacción.
Caja	Registro de movimientos económicos.
Caso de Uso	Acción que modifica o consulta el dominio.
Módulo	Interfaz que utiliza casos de uso.
Evento	Notificación generada por una transacción.

# Lenguaje Ubicuo

Este documento establece el vocabulario oficial del proyecto. Cada término posee un único significado dentro del sistema y deberá utilizarse de forma consistente tanto en la documentación como en el código, con el fin de facilitar la comunicación y evitar ambigüedades durante el desarrollo.
